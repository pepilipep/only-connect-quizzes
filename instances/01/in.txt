## Example customization commands. These may be omitted to use the
## default settings.

# This customizes the "life tokens" displayed when two of the four
# clues in a wall have been solved.
!wall_life_token ꩜

# By default, the questions in the connections, sequences and walls
# rounds are shuffled. These commands disable the shuffling in each
# round separately. They do not take any parameters.
!disable_shuffle_connections
!disable_shuffle_sequences
!disable_shuffle_walls

# Set the time limit for each round type. Setting a time limit to 0
# disables the time limit altogether.
!connections_time_limit 90
!sequences_time_limit 90
!walls_time_limit 150
!vowels_time_limit 90
